IFR Airway Study

Enter answers directly on the chart. Routes are generated from a simple text format so you can swap them out quickly.

How to edit routes

Edit the airwayTextData list in the JavaScript using this format.

POINT1: MLU:VORTAC:087
POINT2: MEI:VORTAC:272
POINT3: none
NAVAIDS: true
NAVAID1: STUEE:false
NAVAID2: none
SEG1: 15
SEG2: 83
TOTALDISTANCE1: 98
TOTALDISTANCE2: none
ROUTENAME: V417

For a middle major point, use two radials on POINT2 like:
POINT2: MEI:VORTAC:272:098
