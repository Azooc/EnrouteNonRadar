IFR Airway Study

This version is data-driven so you can change routes easily. Each airway is built from points and segments. It supports major fixes, intermediate navaids, sub-mileages, total mileages, airway labels, answer checking, answer reveal, and route drawing based on radial degrees.

How to edit routes

Edit the airwayTextData list in the JavaScript using your simple format.

Example:

POINT1: MLU:VORTAC:087
POINT2: MEI:VORTAC:272
POINT3: none
NAVAIDS: true
NAVAID1: STUEE:false
SEG1: 15
TOTALDISTANCE1: 98
ROUTENAME: V417

For a middle major point, use two radials on POINT2 like:
POINT2: MEI:VORTAC:272:098
